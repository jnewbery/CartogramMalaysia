# CartogramMalaysia

This is a Cartogram of Malaysia using shawnbot's [JavaScript implementation](https://github.com/shawnbot/d3-cartogram/) of [an algoritm to construct continuous area cartograms](http://chrisman.scg.ulaval.ca/G360/dougenik.pdf), by James A. Dougenik, Nicholas R. Chrisman and Duane R. Niemeyer, Â©1985 by the Association of American Geographers. It uses [d3](http://github.com/mbostock/d3) for rendering and [TopoJSON](http://github.com/mbostock/topojson) both for writing and reading topological JSON geodata.
